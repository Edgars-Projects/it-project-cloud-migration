# IT Project Management Simulation – Cloud Migration Project

This project simulates the planning phase of an IT project involving the migration of organisational records from paper-based storage to a secure cloud-based system.

The objective is to demonstrate practical IT project management skills including project planning, stakeholder analysis, risk management, and project scheduling.

## Project Objectives

- Improve data accessibility and security
- Reduce reliance on paper-based records
- Implement a secure cloud storage system
- Improve operational efficiency

## Project Deliverables

- Project Charter
- Stakeholder Analysis
- Risk Register
- Project Timeline
- Lessons Learned Documentation

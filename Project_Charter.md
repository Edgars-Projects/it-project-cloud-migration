# Project Charter

## Project Title
Cloud Migration for Organisational Records

## Project Purpose
The purpose of this project is to migrate existing paper-based records to a secure cloud storage platform in order to improve accessibility, data security, and operational efficiency.

## Project Scope

In Scope:
- Digitisation of paper records
- Cloud storage implementation
- Staff training on the new system

Out of Scope:
- Replacement of existing IT hardware
- Development of custom software

## Key Deliverables

- Cloud storage platform setup
- Digitised records repository
- Staff training materials

## Project Success Criteria

- All records successfully digitised
- Secure cloud access implemented
- Staff able to access records remotely

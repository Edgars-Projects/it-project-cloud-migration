# Risk Register

| Risk ID | Risk Description | Likelihood | Impact | Mitigation |
|--------|-----------------|-----------|--------|-----------|
| R1 | Data loss during migration | Medium | High | Implement backup procedures |
| R2 | Staff resistance to new system | Medium | Medium | Provide training and support |
| R3 | Security vulnerabilities in cloud system | Low | High | Implement strong access controls |
| R4 | Migration delays | Medium | Medium | Monitor progress and allocate resources |
| R5 | Cloud service outage | Low | High | Implement backup access procedures |
